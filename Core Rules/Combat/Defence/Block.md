Shields are defensive tools used to redirect, absorb, or prevent incoming strikes. Each shield type defines how you defend: some favour agility, others durability, and others area control.
Shields modify Block and Parry but do not affect melee damage unless specified.
Each shield has:
- A **Block Die** — how much damage it can prevent when Blocking.
- A **Block Speed Modifier** — added to your weapon's base Speed when Blocking.
- A **Parry Modifier** — added to your weapon's base Speed when Parrying.
- An **Evasion Modifier** — large shields make you easier to hit.

## Buckler
A small fist-shield used to deflect rather than absorb.

|Property|Value|
|---|---|
|Block Die|1d4|
|Block Speed Modifier|+2|
|Parry Modifier|+2|
|Evasion Modifier|None|
|Weight / Bulk|Light — does not restrict movement.|

## Heater
A balanced defensive shield offering reliable protection.

|Property|Value|
|---|---|
|Block Die|1d8|
|Block Speed Modifier|+0|
|Parry Modifier|+0|
|Evasion Modifier|None|
|Weight / Bulk|Medium — can be used in one hand without major restriction.|

## Pavise
A massive shield used to cover the body and control space.

|Property|Value|
|---|---|
|Block Die|1d12|
|Block Speed Modifier|–2|
|Parry Modifier|–4|
|Evasion Modifier|–2|
|Weight / Bulk|Heavy — may restrict movement and positioning in tight spaces.|
